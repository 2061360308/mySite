## 大纲

>  简单罗列不会细说

Js基础的知识点大概包括有:

- 基本数据类型
  
  - Integer 
  - String
  - undefined
  - null
  - bool

- 运算符
  
  - 数学运算符
    
    ```
    +
    -
    *
    /
    %
    ```
  
  - 比较运算符
    
    ```
    > 
    <
    >=
    <=
    ==
    ===
    ```

- 语句
  
  - 条件
    
    ```
    if(){}
    
    if(){}else{}
    
    if(){}else if(){}else{}
    ```
  
  - 循环
    
    ```
    for(初始值;判断条件;更改初始值){}
    
    forEach(function(item, index){})  // 一般用来遍历数组
    
    for(var i in obj){}  // 一般用来遍历对象
    ```

- 函数
  
  ```
  function(){}
  ```

- 数组

- 对象

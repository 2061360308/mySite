# 简单使用

vuex 的使用大致分为下面几步:

  1. 使用 Vue 的插件引入函数 use 使用 Vuex Vue.use(Vuex);
  2. 使用 Vuex 实例对象的 Store 方法创建一个 store 对象 var store = new Vuex.Store({})
  3. 在 vue 实例化对象时传入 store 来使用 store
